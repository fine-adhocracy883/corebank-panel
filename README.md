## 🧯 Basic troubleshooting

### The app does not open
- Check that the file finished downloading
- Right-click the file and choose Run as administrator
- Reboot your PC and try again
- Download the file again if it looks broken

### Windows blocks the file
- Open the file details
- Click More info if shown
- Choose Run anyway only if you trust the source

### The panel shows no data
- Check your internet connection
- Confirm the API key is correct
- Make sure the date range is right
- Refresh the page

### Notifications do not arrive
- Check notification settings
- Confirm the destination address or channel
- Make sure the app can reach the network
- Test the notification again

### Webhooks do not work
- Check the webhook URL
- Make sure the event type is correct
- Review the logs
- Confirm the endpoint is active
